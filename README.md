# 🎵 Meu Player de Música

Projeto que integra **YouTube Data API v3 (Google)** para buscar músicas e **CyberHost API** para baixar áudio/vídeo.  
Frontend em HTML/JS e backend em Node.js com Express.

---

## 🚀 Funcionalidades
- 🔎 Buscar músicas no YouTube por nome/artista
- ▶ Assistir vídeo diretamente no YouTube
- 🎧 Escutar áudio (MP3)
- 📹 Baixar vídeo (MP4)
- ⏯️ Controles de play/pause
- 📥 Download direto via CyberHost API
